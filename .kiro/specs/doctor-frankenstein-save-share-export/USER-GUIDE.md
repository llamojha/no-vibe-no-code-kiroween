# Doctor Frankenstein - User Guide

## Overview

Doctor Frankenstein combines random technologies to generate innovative startup ideas using AI.

## Features

### 1. Generate Ideas
- **Tech Companies Mode:** Combine companies like OpenAI, Stripe, Cloudinary
- **AWS Services Mode:** Combine AWS services like Lambda, S3, DynamoDB

### 2. Save Ideas
- Requires authentication
- Stores complete analysis
- Generates unique shareable ID

### 3. Share Ideas
- Copy link to clipboard
- Works without authentication
- Format: `https://domain.com/doctor-frankenstein?savedId=UUID`

### 4. Export Ideas
- **PDF:** Full formatted report with idea name in filename
- **Markdown:** Structured document with frontmatter
- **JSON:** Programmatic data access

### 5. Dashboard
- View all saved ideas
- Delete unwanted ideas
- Quick navigation to ideas
